# Kiddio

Kiddio is an AI-powered parenting companion web app that helps parent navigate parenting with personalized information, practical tips, and a caring heart.

## Tech Stacks
- Frontend & Backend = Next.js
- Database = PostgreSQL (Supabase)
- ORM = Prisma v7
- AI = Google Gemini API
- Auth = JWT
